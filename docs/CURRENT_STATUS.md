## Известные проблемы и риски

### Риск 1 — Ozon меняет DOM
**Вероятность:** Высокая (обновления раз в 1–3 месяца)  
**Решение:** Парсеры пишем с множественными селекторами + fallback. Мониторинг через error tracking.

### Риск 2 — Ozon блокирует фоновые запросы поиска
**Вероятность:** Средняя (rate limiting)  
**Решение:** Добавить задержку 2–3 сек перед запросом, не делать больше 3 запросов подряд. В v2 — прокси-сервер.

### Риск 3 — Shadow DOM конфликты
**Вероятность:** Низкая  
**Решение:** Полная изоляция через Shadow DOM, собственные шрифты, не используем глобальные стили.

### Риск 4 — CSP Ozon блокирует наш скрипт
**Вероятность:** Низкая (Chrome Extension content scripts обходят CSP сайта)  
**Решение:** Это особенность MV3 — content scripts имеют привилегированный доступ.
