## Что это

**Pomogator.ai** — Chrome-расширение для продавцов на маркетплейсах.  
Анализирует публичную карточку товара конкурента на Ozon прямо в браузере и выдаёт оценку качества карточки по 9 блокам (100 баллов максимум).

Целевая аудитория: продавцы на Ozon и WB, которые хотят понять почему карточка конкурента стоит выше и что улучшить у себя.

---

## Как работает (технически)

1. Пользователь открывает страницу товара на `ozon.ru`
2. Chrome-расширение (content script) автоматически активируется
3. Content script парсит DOM страницы и собирает данные
4. Sidebar-панель расширения показывает оценку карточки по блокам
5. Для поиска позиции в выдаче — background script открывает фоновую вкладку `ozon.ru/search/?text=КЛЮЧ`, парсит результаты, закрывает вкладку

**Данные берём только с публичной страницы товара.** Доступ к ЛК продавца не нужен и не используется.

## Блок "Место в выдаче" — как работает

1. Из названия товара извлекаем топ-3 ключевых слова (существительные, словосочетания, TF-IDF)
2. Background script открывает `ozon.ru/search/?text=КЛЮЧ` в фоновой (невидимой) вкладке
3. Content script на вкладке поиска ищет артикул текущего товара
4. Фиксируем позицию (или "не найден в топ-36")
5. Вкладка закрывается — пользователь ничего не видит

**Регион:** определяем косвенно по сроку доставки ("завтра" = локальный склад). Cookie региона в v2.

## Что НЕ делаем в MVP

- Не анализируем качество фото (нужен AI)
- Не показываем контент-рейтинг (только в ЛК продавца)
- Не сравниваем цену с другими МП
- Не отслеживаем динамику позиций
- Не работаем с ЛК продавца вообще
- Не поддерживаем WB (только Ozon в MVP)

## Ключевые решения принятые в дизайне

- **Стиль:** тёмная тема как основа, неоновые акценты, дизайн 2025–2026
- **Переключение тем:** тёмная / светлая
- **Шрифты:** не Inter/Roboto — выбрать что-то характерное
- **Анимации:** появление блоков, прогресс-бары, hover-эффекты
- **Адаптивность:** работает на всех размерах экрана

## Конкуренты для вдохновения (но не копирования)

- Маяк (Chrome-плагин для WB/Ozon) — самый близкий конкурент
- MPStats — глубокая аналитика, но дорого и сложно
- MarketGuru — доступно, но скучный интерфейс

**Наше отличие:** работает прямо на странице конкурента, бесплатный MVP, WOW-дизайн.
